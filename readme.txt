Requirements:
-	ESP IDF version: 	5.2.1
- 	ESP LVGL Port version:	1.4.0
-	LVGL version:		8.3.11

For installation:
- Install ESP IDF Version
- Add ESP LVGL Port as dependency
or 
- Download LVGL Port and add as "esp_lvgl_port" under your install location of the idf
for instance: **\frameworks\esp-idf-v5.2.1\components\esp_lvgl_port
